Finding Numbers in a Haystack

This program was written as an assignment for “Using Python to Access Web Data” class 
by University of Michigan and Coursera

https://www.coursera.org/learn/python-network-data/home/welcome
 
The program  read through and parse a file with text and numbers and then extract all the numbers in the file and compute the sum of the numbers. The sum is printed out.
